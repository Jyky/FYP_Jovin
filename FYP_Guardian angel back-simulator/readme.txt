1) FYP_Guardian angel contains the Java code for the task allocation algorithm and front-end simulator
2) 
FYP_Guardian angel back-simulator contains the Python code for the back-end simulator which provides the location information to the front-end simulator

3) To run the front end simulator, make sure that all 3 python files in the folder FYP_Guardian angel back-simulator are running first. 
4) Next, run the initializeGuardianAnngel.java file in FYP_Guardian angel folder, and select Floorplan.png located in FYP_Guardian angel back-simulator folder.
